About BioDocker

Welcome Friends to the BioDocker Docs!


BioDocker an open source and community-driven framework which provides system-agnostic executable environments for bioinformatics software with special focus on proteomics, genomics, metabolomics and transcriptomics. BioDocker is based on the popular Docker framework that allows software to be installed and executed under an isolated and controllable environment.

BioDocker Containers provides all the containers maintained by the BioDocker community and ready to be use by the community.

BioDocker Registry is hosted registry of all biodocker images that are ready to be use. BioDocker Registry provides an easy way to find a container.

BioDocker SandBox is used to host all the new software, broken containers. BioDocker SandBox is a GitHub repository and the users can ask for a new container by creating a new issue . Also developers can contribute with the group by generating a container for the requested software.

Docker Specification define a set of guidelines and rules to contribute with BioDocker. The Specifications provides an standarize way of generating a container and contribute with BioDocker Project.
